    ext2viewer

    О программе

    Программа позволяет просматривать файловую систему ext2.
    Основные возможности:
    - просмотр разделов ext2fs на дискетах и жестких дисках;
    - копирование файлов и каталогов с ext2fs;
    - просмотр файлов;
    - высокая скорость копирования.

    Для просмотра разделов ext2fs на жестких дисках в Windows NT/2000 необходим запуск программы с правами администратора:
    1) запуск из командной строки: runas /u:Administrator ext2viewer.exe;
    2) запуск из Проводника: удерживая Shift, кликнуть правой кнопкой по программе, затем выбрать в меню пункт Run As.
    Эти два способа работают только в Windows 2000, в Windows NT 4.0 необходимо войти в систему под логином администратора.
